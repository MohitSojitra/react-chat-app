
# chatily ( React-chat-app )




 ## Technology Which I Used
   ***Backend***
   
 - Nodejs
 - Express
 - Socket io

 ***Frontend***
 

 - React Js
 - Material Ui

<br />

 ## Step to Start Project
 

 
**For Backend**
 1. `git clone https://github.com/MohitSojitra/react-chat-app.git`
 2. `cd server-chat-app`
 3. `yarn`
 4. `yarn start`

**For Frontend**

 1. `cd client-chat-app`
 2. `yarn`
 3. `yarn start`
